AbstractEntity

Represents any documentable PHP language symbol, such as classes, functions, etc.

The child classes hold all attributes that are used for generating documentation for each symbol, while this class has all common attributes such as name (you can't document something anonymous).

Each child class defines handle*Tag methods for each type of tag that is supported by that class. If a child class does not define such a method and a tag exists then the respective method of this class gets called and either it is a generic tag that can be applied to all entities, or it logs an error that this tag was unexpected.

Fields
public entityType: string

A user-friendly name for the current symbol class, eg "function".

public longDescription: string

Any description in the phpDoc comment after the short description.

public name: string

The symbol's name in the source code.

public shortDescription: string

The first line of description appears in listings etc.

public sourceLocation: SourceLocation

The file and line the symbol appeared in.

protected docBlock: DocBlock

Parsed phpDoc comment.

Constructor
__construct(string type, string name, SourceLocation sourceLoc, DocBlock docBlock):

Construct an entity.

This constructor is supposed to be overloaded, and then internally called with the first argument being the entityType, instead of having the caller of this class have to pass it manually.

Methods
public handleParamTag(ParamTag paramTag): void

Handle a param tag, logs an error if not overloaded.

public handleReturnTag(ReturnTag returnTag): void

Handle a return tag, logs an error if not overloaded.

public handleVarTag(VarTag varTag): void

Handle a var tag, logs an error if not overloaded.

protected logUnexpectedTag(string tag): void

Log that an unexpected tag was found (ie, a handle*Tag method was not overloaded, and was called).

protected setDocBlock(DocBlock docBlock): void

Change attributes of this symbol according to documentation.
