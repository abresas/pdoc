PHPClass

PHP Language Classes.

A class has its own documentation, as well as methods and properties. The constructor is treated separately from methods.

Fields
public constructor: PHPMethod | null

The constructor, an empty one assumed if not defined.

public extends: string | null

Name of parent class, if applicable.

public methods: PHPMethod[]

The methods of the class (excluding constructor).

public namespace: string

Name of namespace this class belongs to.

public parentClass: PHPClass | null

Parent class object, if applicable.

public properties: PHPProperty[]

The properties of the class.
